1. Click on one of the shape buttons to add a shape to the canvas.
  2. You can use the colored buttons to change the shape's color.
    • Scroll to see all the controls available.
  3. The buttons at the bottom allow you to rotate and change the shape's size.
  4. Lastly, you can drag a shape to the trash can if you'd like to get rid of it.
